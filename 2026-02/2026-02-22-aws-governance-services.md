# AWS Governance Services

## What You'll Learn
AWS Control Tower vs. License Manager
and the differences between AWS Organizations.

## Importance
Multi-account governance is essential in enterprise AWS environments.

## Key Points
- Organizations: Group and manage multiple AWS accounts.
- Control Tower: Automatically apply governance rules to accounts.
- Preventive Guardrails: Block violations.
- Detective Guardrails: Detect and alert on violations.
- License Manager: Automate license compliance, prevent human error, and reduce management costs.
