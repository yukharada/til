# AWS Security Services

## What
Learned the differences between AWS security services:
Inspector, GuardDuty, KMS, CloudHSM, Certificate Manager

## Why it matters
These services are frequently compared in SAA exam and
technical interviews. Understanding the differences is essential.

## Key Differences
- Inspector: Find vulnerabilities in EC2/ECR
- GuardDuty: Detect threats across AWS account
- KMS: AWS manages encryption keys (multi-tenant)
- CloudHSM: User manages encryption keys (FIPS140-2 Level3)
